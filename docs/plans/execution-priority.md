## Tier 1 — Ship Now

Independent epics with stories ready. No blockers. These form the foundation for everything else.

| Order | Epic | Area | Stories | Why First |
|-------|------|------|---------|-----------|
| **1** | E43-S01/S02/S03 | Fix Failing Tests | 3 | Everything depends on reliable CI. 55+ tests failing. |
| **2** | E51 | Accessibility Phase 1 | 4 | Quick wins (font, density, motion), settings infrastructure used by future features. Marked priority 1 in roadmap. |
| **3** | E50 | Calendar Phase 1-2 | 6 | High user value (iCal feed + study planner), independent. Marked priority 2 in roadmap. |
| **4** | E43-S04/S05/S06/S07 | Foundation Remaining | 4 | Session expiry, completion %, notifications data layer. Unblocks normal app usage. |

**Total: 17 stories**

**Parallelization:** E51 and E50 are fully independent — can run in parallel sessions. E43 test fixes should be first since other epics may add tests.

## Tier 3 — Ship After Validation

These benefit from Tier 1-2 being complete, or have soft dependencies.

| Order | Epic | Area | Stories | Dependencies |
|-------|------|------|---------|-------------|
| **9** | E56 | Knowledge Map Phase 1 | 4 | Independent for Phase 1 (topic scores + heatmap). Phase 3 (decay predictions) needs FSRS upgrade. |
| **10** | E57 | AI Tutoring Phase 1-2 | 5 | Independent for Phases 1-2 (lesson chat + Socratic). Phase 4 needs Knowledge Map for learner profile. |

**Total: 9 stories**

**Soft dependencies:**
- Knowledge Map benefits from quiz data generated by E52 (more data → better scores)
- AI Tutoring benefits from Calendar (study context) but doesn't require it
- Both share Dexie version bumps — coordinate migration numbering

**Validation gate after E57 Phase 2:** Is Socratic mode actually better than direct explanation for this content? User feedback determines if Phase 3+ is worth building.

## Tier 4 — Sync (Major Effort)

The sync infrastructure is a significant investment. Wait until the app is feature-rich and stable before adding cross-device complexity.

| Order | Epic | Area | Stories | Dependencies |
|-------|------|------|---------|-------------|
| **11** | E44 | Sync Pre-Requisites | 4 | Needs E43 test health + exportService working |
| **12** | E45 | Sync Infrastructure | 6 | Needs E44 complete |
| **13** | E46 | P0 Sync Live | 8 | Needs E45 complete |

**Phase 1 Total: 18 stories**

**Decision gate before E44:** "Do I actually use Knowlune on multiple devices?" If not, skip sync entirely and move to Tier 5.

**After E46 ships:** Use sync for 2-4 weeks. Validate it works, find edge cases, confirm value.

| Order | Epic | Area | Stories | Dependencies |
