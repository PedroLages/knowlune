**Date**: 2026-03-06
**Reviewer**: Code Review Agent (Opus)

## What Works Well

1. **Parse-once pattern is well-designed.** `getStreakSnapshot()` reads localStorage once and derives current streak, longest streak, and activity array from a single parse.
2. **DST-safe date handling.** `toLocalDateString()` using Swedish locale and `parseLocalDate()` using `new Date(y, m-1, d)` avoids the UTC-midnight pitfall. `Math.round()` on day diffs accounts for DST hour shifts.
3. **O(n+m) activity counting.** `activityFromLog` pre-builds a `Map<string, number>` in a single pass, replacing the previous O(n*m) filter-per-day approach.
4. **Event dispatch added.** `logStudyAction()` now dispatches `study-log-updated` CustomEvent, fixing AC2.
5. **Standalone functions delegate to parse-once helpers.** No more duplicated streak logic.
