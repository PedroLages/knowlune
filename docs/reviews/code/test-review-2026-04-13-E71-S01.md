## Gaps

### MEDIUM

1. **No test for topic with zero activities** — A topic with `hasFlashcards: false`, `hasQuizzes: false`, `lessons: []` that is fading/weak would generate 0 suggestions. This edge case isn't tested (AC 9 tests empty input and strong topics, not "declining but nothing to suggest").
   - Recommendation: Add test confirming such a topic is excluded from results.

### LOW

2. **No boundary test for urgency score range** — No test verifying urgency stays in `[0, 100]` with extreme inputs (score=0 + stability=0, or score=100 + stability=100).

3. **No test for default `recencyScore` fallback** — When `recencyScore` is undefined, code defaults to `50`. No test covers this path explicitly.

## Verdict

**PASS** — Strong coverage. One missing edge case (zero-activity declining topic) is a gap but not a blocker.
