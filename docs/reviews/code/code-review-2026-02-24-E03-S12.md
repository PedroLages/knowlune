# Code Review: E03-S12 — Code & Media Blocks

**Date**: 2026-02-24
**Story**: E03-S12 — Code & Media Blocks

## What Works Well

1. **Clean Tiptap extension architecture.** `CodeBlockLowlight` properly extended with `ReactNodeViewRenderer` for custom `CodeBlockView`. All four feature extensions registered cleanly in a single `useEditor` call. `StarterKit.configure({ codeBlock: false })` correctly prevents conflict.

2. **Prior review items addressed.** S11 blockers resolved: `Highlight.configure({ multicolor: true })` gone (now plain `Highlight`), `TextStyle`/`Color` dead extensions removed, `@tiptap/extension-link` removed from `package.json`, `handleNoteChange` restored `setNoteText(value)` call, `video://` protocol properly registered.

3. **Consistent codebase conventions.** `cn()` used throughout, `size-4` Tailwind v4 shorthand for icons, `@/` import alias, shadcn/ui `Dialog` for YouTube URL input, `ToolbarButton` maintains `size-11` (44px) touch targets.

## Findings

### Blocker (1)

1. **`src/styles/index.css:120-121`**: Syntax highlighting comment color `oklch(0.60 0.02 250)` has ~3.1:1 contrast ratio against `--color-muted` background. WCAG AA requires 4.5:1. Fix: Darken to `oklch(0.45 0.02 250)` (~5.5:1 contrast).

### High Priority (4)

2. **`src/app/components/notes/NoteEditor.tsx:16,61`**: `createLowlight(common)` imports all 37 languages from `lowlight/common` (~376KB). AC specifies 6 languages (~25KB): JavaScript, TypeScript, Python, CSS, HTML, Bash. Replace with selective imports.

3. **`src/app/components/notes/NoteEditor.tsx:116-118`**: `isValidYoutubeUrl` regex rejects legitimate URLs: `m.youtube.com`, `music.youtube.com`, `youtube.com/shorts/`, `youtube.com/v/`. Learners pasting mobile YouTube links see disabled Insert button with no feedback. Broaden regex.

4. **`src/styles/index.css:166-178`**: Details toggle button styled at 20x20px, well below 44px WCAG touch target. Increase clickable area.

5. **`tests/e2e/story-3-12.spec.ts:127-135`**: First AC2 test only asserts button visibility — never clicks, never verifies image insertion. AC2 drag-and-drop path entirely untested.

### Medium (4)

6. **`src/app/components/notes/NoteEditor.tsx:94-103`**: `formatTimestamp` duplicated in 5+ locations. Extract to shared utility.

7. **`src/styles/index.css:77-128`**: Code block language selector uses raw CSS. Could use Tailwind classes directly on `<select>` in `CodeBlockView.tsx`.

8. **`src/app/components/notes/NoteEditor.tsx:168-181`**: Sequential `for...of` with `await` for multiple image files. Use `Promise.all` for parallel processing.

9. **`tests/e2e/story-3-12.spec.ts`**: No edge case tests: invalid YouTube URL rejection, oversized image rejection, empty code block behavior, 16:9 aspect ratio assertion.

### Nits (3)

10. **`src/app/components/notes/CodeBlockView.tsx:1-2`**: `ReactNodeViewProps` import could merge with line 1 import from `@tiptap/react`.

11. **`src/app/components/notes/CodeBlockView.tsx`**: No `data-testid` on language selector. Tests use fragile `.tiptap select`.

12. **`src/app/components/notes/NoteEditor.tsx:169`**: Editor type could use `import { Editor } from '@tiptap/react'` directly.
