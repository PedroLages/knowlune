- **[src/lib/__tests__/dashboardOrder.test.ts:29] (confidence: 95)**: The `localStorage` mock object is missing `clear()`, `key()`, and `length`. If any test or the imported production module (`dashboardOrder`) calls `localStorage.clear()` (e.g., in `clearDashboardData` or initialization), it will throw `TypeError: localStorage.clear is not a function`, causing a silent test failure or false positive.
Fix: Add the missing methods to the mock object:
```typescript
vi.stubGlobal('localStorage', {
  getItem: vi.fn((key: string) => mockStorage[key] ?? null),
  setItem: vi.fn((key: string, value: string) => {
    mockStorage[key] = value
  }),
  removeItem: vi.fn((key: string) => {
    delete mockStorage[key]
  }),
  clear: vi.fn(() => {
    Object.keys(mockStorage).forEach(k => delete mockStorage[k])
  }),

    return Object.keys(mockStorage).length
  },
  key: vi.fn((index: number) => Object.keys(mockStorage)[index] ?? null),
})
```
