# Code Review: E03-S06 — View Course Notes Collection

**Review Date**: 2026-02-28
**Branch**: `feature/e03-s06-view-course-notes-collection`

## What Works Well

1. Clean grouping logic in CourseNotesTab — useMemo with proper dependencies and fallbacks
2. Proper AlertDialog for delete confirmation satisfying NFR23
3. Good Zustand selector usage in NoteCard (`useNoteStore(s => s.saveNote)`)

## Findings

### Blockers

1. **Stale readOnlyEditor content** (confidence: 95) — NoteCard.tsx:65-74. `useEditor({ content: note.content })` sets content only on initialization. After editing and saving, expanded view shows stale pre-edit content. Fix: Add useEffect calling `readOnlyEditor?.commands.setContent(note.content)` when content changes, or conditionally create editor only when expanded.

2. **Nested interactive elements** (confidence: 92) — NoteCard.tsx:95-130. `div[role="button"]` contains `<Link>` and `<Button>`. WCAG 4.1.2 violation. Fix: Replace inner Button with decorative span.

### High Priority

3. **handleDelete fire-and-forget** (confidence: 90) — NoteCard.tsx:87-89. Shows toast.success before async operation completes. Fix: Type onDelete as Promise<void>, await in try/catch.

4. **handleSave no error handling** (confidence: 85) — NoteCard.tsx:76-85. No try/catch on saveNote. If it rejects, user sees no feedback. Fix: Wrap in try/catch, show toast.error on failure.

5. **Eager Tiptap initialization** (confidence: 88) — NoteCard.tsx:65-74. useEditor runs for every NoteCard even when collapsed. 20+ ProseMirror instances on tab switch. Fix: Defer to expanded state.

6. **useNoteStore without selector** (confidence: 85) — CourseNotesTab.tsx:24. Subscribes to entire store, causing unnecessary re-renders. Fix: Use individual selectors.

### Medium

7. **Duplicated formatTimestamp** (confidence: 92) — NoteCard.tsx:36-39. Shared utility exists at `@/lib/format`. Local version lacks hours support. Fix: Import from shared utility.

8. **No unit tests for NoteCard/CourseNotesTab** (confidence: 90) — stripHtml, formatRelativeDate, view state machine untested.

9. **32x32px touch target** (confidence: 82) — NoteCard.tsx:127. Below 44px WCAG minimum.

### Nits

10. **h-N w-N instead of size-N** (confidence: 88) — Multiple instances in both files.
11. **formatRelativeDate DST edge case** (confidence: 75) — Raw ms division fails at DST boundaries. Use date-fns differenceInCalendarDays.
12. **Tabs defaultValue uncontrolled** (confidence: 70) — CourseDetail.tsx:117. Can't deep-link to Notes tab.
13. **E2E test scope** (confidence: 72) — story-e03-s06.spec.ts:167. Edit button assertion not scoped to notesPanel.
