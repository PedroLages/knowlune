- **Location:** `src/ai/prompts/modes/quiz.ts`, `src/ai/prompts/__tests__/quiz.test.ts:154-162`
- **Detail:** AC2 specifies 100-150 token budget. Estimated actual: ~216 tokens. Test was widened to 80-230 to pass. This is a spec deviation, not a bug — the prompt quality justifies the overshoot.
- **Recommendation:** Update spec AC2 to reflect actual budget (~150-230 tokens) or trim prompt.

**P2. Bloom's advancement logic has edge case (LOW)**
- **Location:** `src/stores/useTutorStore.ts:441`
- **Detail:** `newStreak >= 2 && newStreak % 2 === 0` means advancement happens at streak 2, 4, 6, etc. This means a learner can advance from Remember all the way to Create in a single session with 12 consecutive correct answers — potentially too fast. The spec says "advance after 2 consecutive correct" which is ambiguous about whether advancement should happen repeatedly. Current behavior is reasonable but worth noting.
- **Recommendation:** No change needed unless playtesting reveals issues.
