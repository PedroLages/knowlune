- **Nit** `src/app/hooks/useBookCoverUrl.ts:47` (confidence: 75): The local variable `blobUrl` on line 47 is assigned inside the try block (line 65) but is never read after the try/catch -- the value that matters is written to `previousUrlRef.current` on line 68 directly. The local `blobUrl` declaration is dead code that adds cognitive noise. Fix: remove the `let blobUrl` declaration and inline the await result directly into the state/ref updates. Effort: 1 minute.

```typescript
// Before (line 47 + 64-69):
let blobUrl: string | null = null
// ...
try {

  if (!isCancelled) {

    previousUrlRef.current = blobUrl
  }

// After:
try {
  const url = await opfsStorageService.getCoverUrl(bookId)
  if (!isCancelled) {
    setResolvedUrl(url)
    previousUrlRef.current = url
  }
}
```
