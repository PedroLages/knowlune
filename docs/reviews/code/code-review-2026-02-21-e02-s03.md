## Findings

### Blockers

1. **`VideoPlayer.tsx:288`** — TypeScript fallthrough error. `case ' ':` falls through to `case 'k':` with an ESLint disable comment, but `tsconfig.json` has `noFallthroughCasesInSwitch: true`. `npx tsc --noEmit` confirms `TS7029: Fallthrough case in switch`. Build passes via Vite/esbuild (doesn't type-check) but strict CI would fail. Fix: duplicate the handler for both cases.

2. **Uncommitted working-tree fixes** — The debounce fix (ref-based throttle) and touch target fix (44x44px hit area) exist only as unstaged changes. These must be committed to be part of the branch.

### High Priority

3. **`bookmarks.ts`** — Uses localStorage, not IndexedDB. AC3 explicitly specifies "saved as a bookmark in IndexedDB (`bookmarks` table)". Task 1.1 specifies adding a `bookmarks` table to Dexie.js. The current implementation uses `localStorage.getItem('video-bookmarks')` throughout.

4. **No `beforeunload`/`visibilitychange` save handlers** — Task 2.2 requires saving position on tab close and visibility change. Zero event listeners for these events exist in the codebase.

5. **No unit tests for `bookmarks.ts`** — 8 exported functions with 0 test coverage. No `__tests__/bookmarks.test.ts` file exists.

6. **No duplicate bookmark prevention** — `addBookmark()` blindly pushes without checking `hasBookmarkAt()` (which exists at line 114). Rapid clicks create duplicate entries.

7. **BookmarksList seek button missing `aria-label`** — `BookmarksList.tsx:40-42`. Screen reader announces garbled concatenated text.

8. **BookmarksList delete button not keyboard accessible** — `BookmarksList.tsx:59`. `opacity-0 group-hover:opacity-100` without `focus-visible:opacity-100`.

### Medium

9. **`getProgress()` called on every render** — `LessonPlayer.tsx:44`. Parses entire localStorage JSON on every render cycle without memoization.

10. **Test name misleading** — `story-e02-s03.spec.ts:212` says "persist in IndexedDB" but tests localStorage persistence.

11. **29+ scratch/temp files at project root** — `new_vp_v*.tsx`, `VideoPlayer_fixed.tsx`, `update_vp.py`, etc. Risk accidental commit.

### Nits

12. **`w-2 h-2` could use `size-2`** — Tailwind v4 shorthand for bookmark marker dot.

13. **Inline style for bookmark positioning** — Intentional (dynamic percentage), but comment added in working tree.
