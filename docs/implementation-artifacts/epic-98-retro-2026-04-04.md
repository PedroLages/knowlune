- **Clear root cause identification:** `overflow-hidden` fix addressed both the viewport overflow and the theater mode clipping in one change.

## Lessons Learned

1. **`aspect-video` + `max-h` overflow trap:** When combining `aspect-video` with a `max-h` constraint, the aspect ratio can produce a height that exceeds the max-height, causing visual overflow. Apply `overflow-hidden` to the container to contain the spillover.

2. **Cascading keyboard priority:** For layered UI state (e.g., loop markers + theater mode), ESC should clear the most transient state first before toggling more persistent state. This matches user expectation and avoids accidental dismissal of deeper state.

3. **Polish epics ship fast:** With well-understood components and no new business logic, a polish epic can complete in a single review round with zero blockers. Keeping scope tight (one story per symptom cluster) is the key.
