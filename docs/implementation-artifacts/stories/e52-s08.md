# Story 52.08: Post-Completion Auto-Generate (Optional)

## Story

As a learner,
I want the option to automatically generate a quiz when I finish watching a lesson,
so that I can test my retention at the optimal moment without manually triggering generation.

## Acceptance Criteria

**Given** the learner has enabled "Auto-quiz after lesson" in Settings
**When** a lesson's `contentProgress` is marked as complete
**Then** quiz generation is triggered automatically in the background
**And** a non-blocking notification appears: "Generating quiz for [Lesson Name]..."

**Given** auto-generation is in progress
**When** the quiz is ready
**Then** a sonner toast appears: "Quiz ready for [Lesson Name] -- Take it now?"
**And** clicking the toast navigates to the quiz

**Given** the learner has NOT enabled "Auto-quiz after lesson"
**When** a lesson is completed
**Then** no automatic quiz generation occurs
**And** the "Generate Quiz" button remains available on the lesson page

**Given** auto-generation fails (Ollama offline, timeout)
**When** the error is caught
**Then** a subtle toast appears: "Auto-quiz generation failed -- you can generate manually"
**And** the failure does not block the lesson completion flow

**Given** a quiz already exists for the completed lesson (same transcriptHash)
**When** auto-generation would trigger
**Then** the system skips generation (cache hit)
**And** shows "Quiz already available for [Lesson Name]"
