# Story 52.06: Recommendations UI

## Story

As a learner,
I want to see course recommendations on my dashboard and course detail pages,
so that I can discover new learning content relevant to my interests.

## Acceptance Criteria

**Given** the learner navigates to the Overview dashboard
**When** the page renders
**Then** a "Recommended for You" widget displays 3-5 course cards
**And** the widget uses semantic HTML with proper heading hierarchy
**And** the widget is keyboard navigable and screen reader-friendly

**Given** the learner navigates to a course detail page
**When** the page renders
**Then** a "Similar Courses" section displays 3-4 horizontal course cards
**And** the section uses existing course card components

**Given** a recommendation is displayed
**When** the learner views the card
**Then** each card includes an explanation label (e.g., "Because you completed [Course Name]")
**And** the explanation label uses `text-muted-foreground` design token

**Given** a learner with 0 completed courses
**When** the dashboard renders the recommendation widget
**Then** it displays the fallback message "Complete a few courses to get personalized recommendations"
**And** shows 3 recently imported courses below the message

**Given** the learner clicks a recommended course card
**When** the click is processed
**Then** the learner navigates to the course detail page

**Given** the recommendation hook loads
**When** data is being fetched from Dexie
**Then** a loading skeleton is displayed
**And** the hook returns `{ recommendations, isLoading, error, refresh }`

**Existing files to modify:**

- Course detail page — Add SimilarCourses section

**Existing files to reuse:**
- `src/hooks/useRecommendations.ts` — Hook from S05
- Existing course card components (reuse, do not create new)
- shadcn/ui Skeleton component for loading states

**Design tokens:**
- Explanation label: `text-muted-foreground text-sm`
- Card spacing: 8px base grid (multiples of 0.5rem)
- Section margins: 24px (1.5rem) between major sections
- Card border radius: `rounded-[24px]`
- No hardcoded colors (enforced by ESLint)

**Accessibility:**
- Semantic HTML: `<section>`, `<h2>` for widget headings
- `aria-label` on course card links
- Keyboard navigable (Tab through cards)
- Screen reader-friendly explanation labels

**Dependencies:**
- E52-S05 (Recommendation Engine — provides `useRecommendations` hook)
