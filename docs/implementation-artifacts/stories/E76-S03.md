# Story 76.3: React Email Weekly Digest Template

Status: ready-for-dev

## Story

As a user,
I want to receive a well-designed, branded email showing my weekly learning progress,
so that I can quickly scan my achievements and stay motivated to continue learning.

## Acceptance Criteria

1. **Given** the WeeklyDigestEmail React Email component at `supabase/functions/_shared/email-templates/weekly-digest.tsx`
   **When** rendered with valid props
   **Then** the email includes:
   - Header: Knowlune text logo on brand background (#5e6ad2) with "Your Week in Learning" subtitle
   - Greeting: "Hi {firstName}," with warm subtext
   - This Week's Progress: study time and lessons completed in 2-column grid (24px metric numbers)
   - Streak Status: flame icon + streak count in success color (#3a7553) + best streak
   - Due for Review: card count in destructive color (#c44850) + "Review Now" CTA button
   - Achievements Earned: trophy icon + badge names in gold (#c49245)
   - Course Progress: course names with progress bars (brand fill on muted track) + week change percentage
   - Primary CTA: "Open Knowlune" full-width button
   - Footer: unsubscribe link, manage preferences link, CAN-SPAM text
   **And** all styles are inline CSS (no CSS custom properties)
   **And** layout tables use `role="presentation"` for accessibility

2. **Given** a user with no streak (currentStreak = 0)
   **When** the template renders
   **Then** the Streak Status section is not included in the email

3. **Given** a user with no flashcards due (cardsDueForReview = 0)
   **When** the template renders
   **Then** the Due for Review section is not included in the email

4. **Given** a user with no achievements earned this week
   **When** the template renders
   **Then** the Achievements Earned section is not included in the email

5. **Given** the template file
   **When** the module is imported
   **Then** it exports a `renderWeeklyDigest(props: WeeklyDigestProps): string` function that returns a complete HTML string
   **And** the WeeklyDigestProps interface matches the architecture specification

6. **Given** CTA buttons in the email
   **When** rendered
   **Then** they use the bulletproof table+td button pattern for Outlook compatibility
   **And** touch targets are at least 44px height

7. **Given** the email is viewed on a mobile device (<480px)
   **When** rendered
   **Then** metric columns stack vertically and touch targets increase to 48px

## Dev Notes

### Architecture Decisions

- **AD-9**: Pre-rendered HTML approach — templates authored as React Email TSX but `render()` returns a plain HTML string. Avoids Deno JSX runtime issues.
- All styles must be inline CSS. Email clients do NOT support CSS custom properties — use resolved hex values from Knowlune's theme.

### Design Token Color Map (Inline Values)

| Token | Resolved Value | Usage |
|-------|---------------|-------|
| `--background` | `#faf5ee` | Email body background |
| `--card` | `#ffffff` | Content section background |
| `--foreground` | `#1c1d2b` | Primary text color |
| `--brand` | `#5e6ad2` | CTA buttons, header accent |
| `--brand-foreground` | `#ffffff` | CTA button text |
| `--brand-soft` | `#d0d2ee` | Section header backgrounds |
| `--muted-foreground` | `#656870` | Secondary text, descriptions |
| `--success` | `#3a7553` | Streak active indicator |
| `--gold` | `#c49245` | Achievement badge accent |
| `--destructive` | `#c44850` | Due cards urgency accent |
| `--border` | `rgba(0,0,0,0.07)` | Section separators |

### WeeklyDigestProps Interface

```typescript
interface WeeklyDigestProps {
  userName: string
  weeklyStudyMinutes: number
  lessonsCompleted: number
  currentStreak: number
  bestStreak: number
  cardsDueForReview: number
  achievementsEarned: { title: string; earnedAt: string }[]
  courseProgress: { name: string; percent: number; weekChange: number }[]
  unsubscribeUrl: string
  managePrefsUrl: string
  appUrl: string
}
```

### Conditional Section Logic

| Section | Show When | Skip When |
|---------|-----------|-----------|
| This Week's Progress | `weeklyStudyMinutes > 0` | No study activity |
| Streak Status | `currentStreak > 0` | No active streak |
| Due for Review | `cardsDueForReview > 0` | No cards due |
| Achievements Earned | `achievementsEarned.length > 0` | No new achievements |
| Course Progress | Courses with `weekChange > 0` | No progress changes |

### Typography

- Headings: `'Space Grotesk', system-ui, sans-serif`
- Body: `'DM Sans', system-ui, sans-serif`
- Base: 16px body, 14px secondary, 24px metric numbers
- Line height: 1.5 body, 1.2 headings

### Critical Constraints

- **No CSS variables**: All colors hardcoded as inline hex values.
- **No images for logo**: Use text "Knowlune" (avoids blocked images issue in email clients).
- **Bulletproof buttons**: `<table><tr><td>` pattern — NOT `<a>` with padding (Outlook ignores padding on `<a>`).
- **Max 5 courses** in Course Progress section, sorted by most progress change.
- **role="presentation"** on all layout tables for screen reader compatibility.
- File location: `supabase/functions/_shared/email-templates/weekly-digest.tsx` (Supabase shared module convention).
