# Story 108.2: Format Badges and Delete

## Story

As a reader,
I want to see clear format badges on my books (EPUB, Audiobook, PDF) and be able to delete books from the library,
so that I can quickly identify book types and manage my collection.

## Acceptance Criteria

- **AC-1**: Each book card and list item shows a format badge indicating the book type: "EPUB", "Audiobook", or "PDF"
- **AC-2**: Format badges use distinct icons and colors for each format (e.g., BookOpen for EPUB, Headphones for Audiobook, FileText for PDF)
- **AC-3**: Format badges are accessible with ARIA labels (e.g., `aria-label="EPUB format"`)
- **AC-4**: Book context menu includes a "Delete" option that prompts a confirmation dialog before deletion
- **AC-5**: Delete confirmation dialog shows the book title and warns that OPFS files will be permanently removed
- **AC-6**: On confirmation, book is removed from Dexie and associated OPFS files are cleaned up
- **AC-7**: After deletion, the book disappears from the library view immediately (no stale state)
- **AC-8**: Format badges and delete work in both grid and list views

## Design Guidance

- **FormatBadge**: Small pill-style badge (similar to `BookStatusBadge` pattern from `src/app/components/library/BookStatusBadge.tsx`)
- **Position**: Top-right of BookCard cover image, left-aligned in BookListItem metadata row
- **Colors**: Use distinct design token backgrounds — e.g., `bg-brand-soft` for EPUB, `bg-warning/10 text-warning` for Audiobook, `bg-muted text-muted-foreground` for PDF
- **Delete dialog**: Use `AlertDialog` with destructive action variant on the confirm button
