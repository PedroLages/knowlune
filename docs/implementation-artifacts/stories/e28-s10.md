# Story 23.10: Transcript Panel with Search & Click-to-Seek

## Story

As a learner,
I want to see synchronized transcripts alongside YouTube videos, search within them, and click to jump to specific moments,
so that I can study video content more efficiently.

## Acceptance Criteria

1. TranscriptPanel displays transcript cues alongside the video, with currently active cue highlighted based on player's current time
2. Panel auto-scrolls to keep active cue visible
3. Click on any transcript segment -> player seeks to that timestamp, playback continues
4. Search input: matching segments highlighted, results within 100ms, filters/highlights matching cues while preserving full transcript context
5. Global search: YouTube transcript fullText matches appear in search results, each linking to specific video at matching timestamp
6. No transcript: empty state "No transcript available for this video" + Whisper setup hint
7. Transcript still pending: loading skeleton with "Fetching transcript..."
8. Reactive updates via useYouTubeTranscriptStore -- no page refresh needed
9. Keyboard navigation: segments focusable via Tab, Enter seeks to timestamp
10. WCAG 2.2 AA requirements met
