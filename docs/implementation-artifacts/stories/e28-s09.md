# Story 28.9: YouTube IFrame Player & Progress Tracking

## Story

As a learner,
I want to watch YouTube videos within Knowlune with the same progress tracking as local videos,
so that my study streaks, session logging, and completion tracking work seamlessly for YouTube courses.

## Acceptance Criteria

1. YouTubeLessonPlayer page loads at route `youtube-courses/:courseId/lessons/:lessonId`
2. YouTube video plays via embedded `react-youtube` IFrame player
3. Player UI matches local video player layout (video left/top, notes right/bottom)
4. Player polls `getCurrentTime()` every 1s, stores position in `progress` table using compound key `[courseId+videoId]`
5. Resume from last position on next visit
6. Auto-complete when >90% watched
7. Manual completion toggle (Not Started / In Progress / Completed)
8. Study session logging captures duration and content
9. COEP set to `credentialless` (not `require-corp`) for YouTube IFrame + WebLLM coexistence
10. CSP headers: `frame-src` includes youtube.com and youtube-nocookie.com, `img-src` includes i.ytimg.com and img.youtube.com, `connect-src` includes googleapis.com/youtube/
11. Offline placeholder: "Connect to the internet to watch" with WifiOff icon
12. YouTube course detail page at `youtube-courses/:courseId` with chapter structure, per-video progress, overall completion
