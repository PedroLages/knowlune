## Implementation Notes

**Root cause:** All 5 test files broke because components/stores were refactored during E87-E103 but their test mocks weren't updated. The tests themselves are structurally sound — only the mock shapes need alignment.

**Files to modify:**
- `src/app/components/figma/__tests__/ImportWizardDialog.test.tsx`
- `src/app/pages/__tests__/Courses.test.tsx`
- `src/stores/__tests__/useFlashcardStore.test.ts`
- `src/stores/__tests__/useReviewStore.test.ts`
- `src/stores/__tests__/useSessionStore.test.ts`

**Approach:** Run each test file individually, read error output, update mocks to match current signatures. Do NOT rewrite tests — only update mock shapes.
