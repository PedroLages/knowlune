# Story 108.4: Audiobook Settings Panel

## Story

As an audiobook listener,
I want a settings panel to configure default audiobook playback preferences (speed, skip silence, auto-sleep timer),
so that my preferred listening experience is consistent across all books without manual adjustment each time.

## Acceptance Criteria

- **AC-1**: Audiobook settings section is accessible from the Library page settings area or a dedicated panel accessible from the audiobook player
- **AC-2**: Default playback speed can be set (0.5x to 3x, 0.25x increments) — applies to new audiobook sessions
- **AC-3**: "Skip Silence" toggle — when enabled, automatically skips silent sections during playback
- **AC-4**: Default sleep timer duration can be configured (off, 15min, 30min, 45min, 60min, end-of-chapter)
- **AC-5**: "Auto-bookmark on stop" toggle — when enabled, creates a bookmark at the current position when playback stops
- **AC-6**: Settings are persisted to localStorage (user preferences, not per-book) and survive app restarts
- **AC-7**: Per-book speed override is preserved (changing global default does not overwrite a per-book speed already set)

## Design Guidance

- **Panel**: Use `Sheet` component (slide-in from right) for settings, consistent with other settings panels
- **Speed slider**: Use existing `SpeedControl.tsx` as a reference pattern, but make it configurable (not hardcoded)
- **Toggles**: Use shadcn `Switch` component with labels
- **Sleep timer presets**: Use `RadioGroup` with descriptive labels
- **Visual hierarchy**: Group related settings (Playback, Sleep Timer, Bookmarks) with section headers

## Implementation Notes

**Key files to modify:**
- `src/app/components/audiobook/AudiobookRenderer.tsx` — add settings gear icon + Sheet trigger
- `src/app/components/audiobook/SpeedControl.tsx` — reference for speed UI pattern
- `src/app/hooks/useSleepTimer.ts` — reference for sleep timer integration
- `src/stores/useAudioPlayerStore.ts` — per-book speed storage (already exists)

**New files:**
- `src/stores/useAudiobookPrefsStore.ts` — global audiobook preferences (Zustand + localStorage)
- `src/app/components/audiobook/AudiobookSettingsPanel.tsx` — settings UI

**Existing patterns to follow:**
- `useReadingGoalStore.ts` — localStorage persistence pattern with Zustand
- `usePomodoroPrefsStore` — user preferences store pattern
- `SpeedControl.tsx` — speed slider UI pattern
- `SleepTimer.tsx` — sleep timer UI pattern

**Per-book speed vs global default**: The `useAudioPlayerStore` already stores per-book playback speed. The new global default should only be applied when a book has no per-book speed set yet. Check `useAudioPlayerStore.getSpeedForBook(bookId)` — if undefined, use global default.

**Skip silence implementation**: This is a non-trivial audio processing feature. Consider using Web Audio API's `AnalyserNode` to detect silence (< threshold for > N ms). Start with a simple implementation, flag as future enhancement if too complex.
