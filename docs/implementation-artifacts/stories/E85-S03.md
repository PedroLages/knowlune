# Story 85.3: Highlight List Panel

Status: done

## Story

As a learner,
I want to view all highlights for a book in a consolidated panel,
so that I can review my marked passages and navigate directly to them.

## Acceptance Criteria

1. **Given** the user is reading an EPUB **When** the user selects "Highlights" from the reader header menu **Then** a Sheet slides in from the right showing all highlights for the current book **And** the panel header shows the highlight count (e.g., "Highlights (24)") and a close button **And** color filter pill buttons (All, yellow, green, blue, pink) allow filtering by color

2. **Given** the highlight list panel is open **When** the user views highlight cards **Then** each card shows: color indicator dot (size-3 rounded-full), text preview (3-line clamp), chapter and date, note (if exists, italic with StickyNote icon), and flashcard link indicator (Layers icon in brand color if linked, muted-foreground if not)

3. **Given** the highlight list panel is open **When** the user taps a highlight's text **Then** the reader navigates to that highlight's location in the EPUB using the stored CFI **And** the highlight list panel closes

4. **Given** the highlight list panel is open **When** the user taps the flashcard icon on a highlight without a linked flashcard **Then** the ClozeFlashcardCreator opens with that highlight's text pre-loaded

## Dev Notes

### Architecture Constraints

- **Highlight data** comes from `useHighlightStore.highlights` which was populated by `loadHighlightsForBook(bookId)` on reader open (E85-S01).
- **Color filter** updates `useHighlightStore.colorFilter` — the highlight list is a derived/filtered view of the same store.
- **Navigation via CFI**: `rendition.display(cfiRange)` navigates to a specific CFI. This is the same API used by the TOC (E84-S02). Access rendition ref from EpubRenderer context.
- **Sheet from right** is consistent with highlight panels in reading apps. Use shadcn `Sheet` with `side="right"`.

### Key Dependencies

- useHighlightStore with highlights array and colorFilter (E85-S01)
- ReaderHeader dropdown with "Highlights" menu item (E84-S01)
- Rendition ref for CFI navigation (E84-S01)
- ClozeFlashcardCreator (E85-S04) — stub the trigger if not yet implemented

### Accessibility

- Sheet: focus trap when open, `aria-label="Highlights panel"`
- Color filter pills: `role="tablist"` with `role="tab"` on each pill
- Highlight cards: `role="article"` with descriptive `aria-label`
- Navigation on tap: announce page change via `aria-live` region
