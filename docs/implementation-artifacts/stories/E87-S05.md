# Story 87.5: Mini-Player and Media Session Integration

Status: ready-for-dev

## Story

As a learner,
I want a persistent mini-player bar when navigating away from the audiobook page and OS-level media controls,
so that I can continue listening while browsing other parts of Knowlune and control playback from my lock screen.

## Acceptance Criteria

**Given** an audiobook is playing
**When** the user navigates away from the BookReader page
**Then** a mini-player bar appears fixed at the bottom of the screen (above any existing UI)
**And** the mini-player shows: cover thumbnail (size-10), title/chapter text, play/pause button, skip back/forward buttons, speed indicator, expand button (ChevronUp), and a thin progress bar
**And** on mobile, the mini-player shows only play/pause; tapping the bar expands to full player

**Given** an audiobook is playing
**When** the system checks Media Session API support
**Then** the Media Session API is configured with: title, artist, album (book title), artwork (cover), and action handlers for play, pause, previoustrack, nexttrack, seekbackward, seekforward
**And** the user can control playback from their OS lock screen, notification center, or Bluetooth headset controls

## Dev Notes

### Architecture Patterns

- **Singleton audio**: The audio element MUST persist across route changes. Two approaches:
  1. Mount `<audio>` in Layout component outside `<Outlet />`
  2. Use a module-level `Audio()` instance in useAudioPlayer (not tied to React lifecycle)
  Option 2 is recommended — simpler, no DOM element management.
- **Layout integration**: `src/app/components/Layout.tsx` already manages sidebar + header + content area. Add mini-player as a sibling to the Outlet, conditionally rendered.
- **Store-driven**: Mini-player reads all state from `useAudioPlayerStore` — no prop drilling.

### UX Design (UX-DR13)

- Mini-player: fixed bottom, persistent across pages
- Cover (size-10), title/chapter, play/pause, skip 15s/30s, speed, expand button, thin progress bar
- Mobile: simplified — play/pause only, tap to expand
- Mini-player height: 64px. Add `pb-16` to main content when visible.

### Media Session API

```typescript
if ('mediaSession' in navigator) {
  navigator.mediaSession.metadata = new MediaMetadata({
    title: chapterTitle,
    artist: bookAuthor,
    album: bookTitle,
    artwork: [{ src: coverUrl, sizes: '256x256', type: 'image/jpeg' }]
  })
  navigator.mediaSession.setActionHandler('play', () => play())
  navigator.mediaSession.setActionHandler('pause', () => pause())
  navigator.mediaSession.setActionHandler('seekbackward', () => skipBack(15))
  navigator.mediaSession.setActionHandler('seekforward', () => skipForward(30))
  navigator.mediaSession.setActionHandler('previoustrack', () => prevChapter())
  navigator.mediaSession.setActionHandler('nexttrack', () => nextChapter())
}
```
