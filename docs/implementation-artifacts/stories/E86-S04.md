# Story 86.4: PKM Export for Book Highlights

Status: ready-for-dev

## Story

As a learner,
I want to export my book highlights in Obsidian and Readwise-compatible formats,
so that I can integrate my reading notes into my personal knowledge management workflow.

## Acceptance Criteria

**Given** the user has highlights for one or more books
**When** the user initiates a PKM export (from existing export infrastructure)
**Then** book highlights are included in the export alongside course notes
**And** the Obsidian format exports each book as a markdown file with: book title as H1, author as metadata, highlights grouped by chapter, each highlight showing the quoted text, annotation (if exists), color, and date
**And** the Readwise format exports highlights as CSV with columns: title, author, highlight text, note, location, color, date
**And** export includes a count summary: "Exported {n} highlights from {m} books"

### Obsidian Markdown Format

```markdown
# The Pragmatic Programmer
Author: David Thomas, Andrew Hunt

## Chapter 3: The Basic Tools

> "Care about your craft. Why spend your life developing software unless you care about doing it well?"

Note: Core philosophy of the book
Color: yellow | 2026-03-29

> "Don't live with broken windows."

Color: green | 2026-03-29
```

### Readwise CSV Format

```csv
Title,Author,Highlight,Note,Location,Color,Date
"The Pragmatic Programmer","David Thomas, Andrew Hunt","Care about your craft...","Core philosophy","Chapter 3","yellow","2026-03-29"
```
