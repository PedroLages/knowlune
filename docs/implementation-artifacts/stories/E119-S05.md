# Story E119-S05: Registry-Driven Export ZIP

## Story

As a user (data subject exercising Art 15/20 rights),
I want to download a complete ZIP of all my data — every sync table and all Storage bucket media — driven by `tableRegistry.ts`,
so that I receive a portable, complete, and verifiable copy of my data.

## Acceptance Criteria

- **AC-1**: `export-data` Edge Function streams a ZIP containing: `data.json` (all 26 sync tables, LWW server-truth merged with local Dexie local-only rows tagged `_origin: 'local'`), `media/<bucket>/...` (all user-owned objects from 4 buckets), `README.md` (manifest).
- **AC-2**: README manifest includes: export timestamp, notice version acknowledged, schema version, per-table row counts, per-bucket object counts, contact email.
- **AC-3**: If streaming would exceed 500 MB, Edge Function aborts and returns `{ status: 'too-large', route: 'async' }` — the S06 async path picks this up.
- **AC-4**: Export requires JWT authentication; user ID from JWT is the sole filter (no cross-user data possible via RLS + explicit filter).
- **AC-5**: Export is registry-driven: every table in `tableRegistry.ts` is automatically included; no per-table hardcoding.
- **AC-6**: `src/app/components/settings/MyDataSummary.tsx` updated to call the new Edge Function; legacy `exportAllAsJson` preserved for non-GDPR backup.
- **AC-7**: E2E test: real user can download, unzip, and see their own data in `data.json`.
- **AC-8**: Test: RLS misconfiguration on any table causes export to fail closed with an explicit error (not a silent gap).

## Context

- Depends on S01 for notice version in README manifest.
- Depends on S03 for the registry-iteration pattern (export and delete iterate the same registry).
- `exportService.ts` already has schema-version pattern (`CURRENT_SCHEMA_VERSION = 14`) — preserve it.
- ZIP streaming in Deno: use `std/archive` or esm.sh zip lib.
- 500 MB guard: measure running byte count during iteration; abort when threshold hit.
- RLS check: if a table select returns an error (not empty rows), fail the export with explicit message.

## Out of Scope

- Async export path for > 500 MB (S06).
- Consent-gated data categories (S08 handles consent toggles).
