# Story E109-S04: Annotation Summary View

## Story

As a reader,
I want to view a summary of all my highlights and annotations for a specific book,
so that I can quickly review my key insights and learning from that book.

## Acceptance Criteria

- **AC-1**: From the book context menu in the library, the user can open an "Annotation Summary" view for a specific book
- **AC-2**: The annotation summary page shows highlight statistics (total count, by color category)
- **AC-3**: All highlights for the book are listed with their text, color, and creation date
- **AC-4**: The page shows an empty state when no highlights exist for the book
- **AC-5**: The annotation summary is accessible via a direct route `/annotations/:bookId`
