## Lessons Learned

1. **Singleton audio survives navigation** — Module-level `new Audio()` over DOM-mounted element. Layout Outlet destroys elements on route change.

2. **30s pause threshold for Smart Resume** — Balances context loss (long pause) vs. toggle annoyance (quick pause/play).

3. **Media Session API requires guard** — Always `if ('mediaSession' in navigator)` before use.

4. **ID3 parsing needs only 4KB** — No need for music-metadata npm for basic tag extraction.

5. **Component composition > monolithic** — E87's subcomponent pattern avoided E84's 350+ line component problem.

6. **Empty review gates can mask issues** — All stories passed build+lint but had no human review. Budget review time for E88.
