# Story 24.2: Import Wizard Folder Selection and Course Details

## Story

As a learner,
I want a two-step import wizard with folder selection and course detail editing,
so that I can preview and customize course metadata before importing.

## Acceptance Criteria

- AC1: Dialog opens from the Import Course button on the Courses page.
- AC2: Step 1 shows a folder selection UI with a "Select Folder" button that triggers the directory picker.
- AC3: After scanning, Step 2 shows editable course details (name, video/PDF counts).
- AC4: Import button persists the scanned course with any user overrides (name changes).
- AC5: Back button returns to folder selection; closing the dialog resets wizard state.

- [x] Task 2: Integrate scanCourseFolder and persistScannedCourse from E24-S01 (AC: 2, 4)

## Implementation Notes

- Two-step wizard: Step 1 (folder selection with scanning state) -> Step 2 (course details with editable name).
- Uses scanCourseFolder() for read-only scan, then persistScannedCourse() with optional overrides.

- All design tokens used correctly (bg-brand, variant="brand", text-muted-foreground, etc.).
- resetWizard callback cleans up all state on close.
